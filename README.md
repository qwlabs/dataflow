# Installation

## Maven

```xml

<dependency>
    <groupId>com.qwlabs.dataflow</groupId>
    <artifactId>dataflow-core</artifactId>
    <version>0.1.*</version>
    <type>pom</type>
</dependency>
```
```xml
<dependency>
    <groupId>com.qwlabs.dataflow</groupId>
    <artifactId>dataflow-functions</artifactId>
    <version>0.1.*</version>
    <type>pom</type>
</dependency>
```

## Gradle

```gradle
implementation 'com.qwlabs.dataflow:dataflow-core:0.1.*'
```
